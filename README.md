# ZFC公理系统结构可视化（GitHub Mermaid版）

## 1. ZFC公理相互关系流程图
```mermaid
graph TD
    %% 底层：基础公理（无依赖，定义与起点）
    subgraph 底层：基础公理
        A[外延公理<br/>定义集合相等性]
        B[空集公理<br/>提供初始集合∅]
    end

    %% 中层：构造公理（依赖底层，生成新集合）
    subgraph 中层：构造公理
        C[配对公理<br/>2元素→新集合]
        D[并集公理<br/>集合族→合并元素]
        E[幂集公理<br/>集合A→幂集P(A)]
        F[子集公理<br/>集合+性质→子集]
        G[替换公理<br/>集合+函数→新集合]
        H[无穷公理<br/>断言无限集合N存在]
    end

    %% 顶层：限制公理（约束所有集合，保安全）
    subgraph 顶层：限制公理
        I[正则公理<br/>禁止自属于/无限递降链]
        J[选择公理<br/>集合族→选择函数]
    end

    %% 依赖关系
    A --> C & D & E & F & G & H  %% 外延公理支撑所有构造公理
    B --> C & H                  %% 空集公理支撑配对、无穷公理
    C --> D                      %% 配对公理支撑并集公理（生成集合族）
